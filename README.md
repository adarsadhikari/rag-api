# RAG API with FastAPI

A simple Retrieval-Augmented Generation (RAG) API built with FastAPI and Neon PostgreSQL database.

## Features

- FastAPI backend
- RAG pipeline support
- Neon PostgreSQL database
- REST API endpoints
- Easy environment configuration

---

## Tech Stack

- Python
- FastAPI
- NeonDB (PostgreSQL)
- SQLAlchemy / Psycopg
- Uvicorn
